public-tools

ブラウザだけで動く便利な web ツール集。

ツール一覧

  • 文字数カウンタ
    文字数・単語数・行数・バイト数をリアルタイム計測。
  • 差分比較
    2 つの文章を行単位で比較してハイライト表示。
  • タイマー / ストップウォッチ
    カウントダウンタイマーとラップ計測付きストップウォッチ。
  • ルーレット
    入力した項目から 1 つをランダムに抽選。
  • カラーピッカー
    HEX / RGB / HSL を相互変換しコピー。
  • JSON 整形
    JSON の整形・圧縮・キーソートとシンタックスハイライト。
  • コード整形 (HTML / CSS / JS)
    HTML・CSS・JavaScript の簡易フォーマッタ（外部依存なし）。
  • Unix time 変換
    タイムスタンプと日時を相互変換、現在時刻も表示。
  • Base64 エンコード / デコード
    UTF-8 文字列と Base64 を相互変換、URL safe にも対応。
  • 正規表現テスター
    JavaScript の RegExp でマッチ確認・置換、ハイライト表示。
  • パスワード生成
    crypto.getRandomValues を使った安全な乱数でパスワードを生成。
  • UUID 生成
    UUID v4 (ランダム) と v7 (時刻ベース) を一括生成。
  • Markdown プレビュー
    Markdown をリアルタイムに HTML 変換、テーブル/タスクリスト対応。
  • QR コード生成
    テキストや URL から QR コードを生成（外部依存なし、PNG 保存可）。
  • URL エンコード / デコード
    パーセントエンコード変換とクエリ文字列のパース／組み立て。
  • ハッシュ計算
    テキスト/ファイルの SHA-1 / 256 / 384 / 512 を一括計算。
  • 画像加工
    リサイズ・JPEG/WebP 圧縮・グレースケール / セピア / 反転・明度/コントラスト調整。
  • 進数変換
    2 / 8 / 10 / 16 進相互変換と AND/OR/XOR/NOT/シフト演算。
  • 時間単位変換
    秒・分・時間・日を相互変換、HH:MM:SS や 1h30m 表記にも対応。
  • 電卓
    四則演算・%・メモリ・税込/税抜・履歴つきの電卓。キーボード対応。
  • 割り勘計算
    合計金額と人数から 1 人あたりを計算、幹事の上乗せ・端数丸め対応。
  • 年齢計算
    生年月日から満年齢・生まれてからの日数・干支・星座・次の誕生日まで。
  • 文字置換
    複数の置換ペアを一括適用。大文字小文字・正規表現・単語境界に対応。
  • 文字種変換
    大小・全半角・ひらカタ・命名規則・並べ替えなど 20+ の変換をワンクリック。
  • 単位換算
    長さ・重さ・温度・面積・体積・速度・データサイズの相互変換。
  • 日付計算
    2 日付の期間計算と日付の加減算。営業日（月〜金）モード対応。
  • BMI 計算
    身長・体重から BMI と肥満度判定、標準体重・美容体重を表示。
  • メモ帳
    複数ノート対応の自動保存メモ帳（ブラウザ内に保存、外部送信なし）。
  • 割合計算
    パーセントに関する 5 パターンの計算（割引・増減率・◯%は何 など）。
  • ダミー画像生成
    幅・高さ・色を指定してプレースホルダー画像を生成（PNG/JPEG/WebP/SVG）。
  • コイントス・サイコロ・じゃんけん
    迷ったときの抽選セット。複数サイコロや TRPG 用 d20 にも対応。
  • TODO リスト
    優先度・期限つきタスク管理。ブラウザに自動保存（外部送信なし）。
  • 西暦・和暦変換
    明治以降の元号と西暦を相互変換、干支・曜日・元号期間も表示。
  • 入学・卒業年計算
    生年月日から小中高大の入学年・卒業年を西暦＋和暦で表示（履歴書用）。
  • テーブルジェネレータ
    セルを編集して Markdown / HTML / CSV / TSV のテーブルを生成。CSV 取り込みも可。
  • CSS Box Shadow ジェネレータ
    スライダで影をリアルタイム編集、複数シャドウ重ね合わせ・プリセット付き。
  • IP アドレス確認
    グローバル IP・ブラウザ環境情報・ローカル IP (WebRTC) を表示。
  • CSS Gradient ジェネレータ
    線形・放射状・円錐グラデーションをスライダで作成、CSS をコピー。
  • HTML 閉じタグチェッカ
    HTML の閉じ忘れ・対応していない閉じタグ・余分な閉じタグを行番号つきで指摘。
  • OGP / メタタグ確認
    URL または HTML から OGP・Twitter Card を抽出し、シェア時の見た目をプレビュー。
  • Lorem Ipsum / ダミー文章 生成
    英文・日本語のダミー文章を段落／文／単語／文字単位で生成。
  • CSS Border Radius ジェネレータ
    4 隅 × X/Y を独立に編集、楕円対応＆プリセット (ブロブ・葉等) 付き。
  • 読了時間 / SEO 文字数チェッカ
    記事の読了時間と SEO 目標バー、長文検出・漢字比率も表示。
  • 目次 (TOC) ジェネレータ
    Markdown / HTML の見出しからリンク付き目次を生成（MD / HTML 出力）。
  • HTML 要素抽出
    記事 HTML から見出し・リンク・画像 alt を一覧化し SEO チェック、CSV 出力対応。
  • キーワード出現頻度
    単語・文字 N-gram の出現頻度と密度をランキング表示。SEO の偏り確認に。
  • 数字 ⇄ 漢数字 変換
    アラビア数字と漢数字（位取り・読み式・大字）の相互変換、文中一括も可。
  • 重複行・重複文 検出
    同じ言い回しが繰り返されていないかを検出。行・文・連続フレーズに対応。

Source: github.com/tiwuofficial/public-tools
