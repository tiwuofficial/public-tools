public-tools

ブラウザだけで動く便利な web ツール集。

ツール一覧

  • 文字数カウンタ
    文字数・単語数・行数・バイト数をリアルタイム計測。
  • 差分比較
    2 つの文章を行単位で比較してハイライト表示。
  • タイマー / ストップウォッチ
    カウントダウンタイマーとラップ計測付きストップウォッチ。
  • ルーレット
    入力した項目から 1 つをランダムに抽選。
  • カラーピッカー
    HEX / RGB / HSL を相互変換しコピー。
  • JSON 整形
    JSON の整形・圧縮・キーソートとシンタックスハイライト。
  • コード整形 (HTML / CSS / JS)
    HTML・CSS・JavaScript の簡易フォーマッタ（外部依存なし）。
  • Unix time 変換
    タイムスタンプと日時を相互変換、現在時刻も表示。
  • Base64 エンコード / デコード
    UTF-8 文字列と Base64 を相互変換、URL safe にも対応。
  • 正規表現テスター
    JavaScript の RegExp でマッチ確認・置換、ハイライト表示。
  • パスワード生成
    crypto.getRandomValues を使った安全な乱数でパスワードを生成。
  • UUID 生成
    UUID v4 (ランダム) と v7 (時刻ベース) を一括生成。
  • Markdown プレビュー
    Markdown をリアルタイムに HTML 変換、テーブル/タスクリスト対応。
  • QR コード生成
    テキストや URL から QR コードを生成（外部依存なし、PNG 保存可）。
  • URL エンコード / デコード
    パーセントエンコード変換とクエリ文字列のパース／組み立て。
  • ハッシュ計算
    テキスト/ファイルの SHA-1 / 256 / 384 / 512 を一括計算。

Source: github.com/tiwuofficial/public-tools
